# ChordsBoy2
ArduBoy Project for musical scales visualization

- 16 Musical Modes
- Grid with 3rd, 5th. 7th. 9th intervals
- Augmented and diminished grid
- Listen scales live!

With hex file for instant upload

# Menu Info
0-ACC: Accordi<br>
1-AUM: Accordi Aumentati<br>
2-DIM: Accordi Diminuiti<br>
3-INF: info<br>
